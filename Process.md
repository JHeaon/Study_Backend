# OperatingSystem

## OperatingSystem 이란?
OperatingSystem 통칭 OS(운영체제)는 컴퓨터 하드웨어(CPU, I/O)를 관리해주면서 동시에 여러개의 Appliction들이 작동 할 수 있는 환경을 제공하는 Software 이다.
